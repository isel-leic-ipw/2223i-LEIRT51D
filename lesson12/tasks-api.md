# Tasks API definition


| Method | Path  | Description |  
|--------|-------------|---------------------------|
| GET    | /tasks      | Obtain the list of tasks  |
| GET    | /tasks/:id  | Obtain a task             |
| DELETE | /tasks/:id  | Delete a task             |
| PUT    | /tasks/:id  | Update a task             |
| POST   | /tasks/     | Create a task             |
